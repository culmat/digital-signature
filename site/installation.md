# Installation

**Digital Signature** is a Confluence Cloud app. It requires a [Confluence Cloud](https://developer.atlassian.com/cloud/confluence/) instance and a Confluence administrator to install it.

## Install from the Atlassian Marketplace

1. Go to **Confluence Settings -> Manage apps** (you need site-admin access).
2. Click **Find new apps** and search for **Digital Signature**.
3. Click **Get app** and follow the prompts.

Once installed, the macro is available to all users on your site immediately.

## Add the macro to a page

1. Open a Confluence page in edit mode.
2. Type `/` to open the macro browser, then search for **Digital Signature**.
3. Insert the macro. An empty panel appears with a prompt to open the macro settings.
4. Click the macro to open its configuration panel and fill in the contract title and text.
5. Save the page.

The signing interface is visible to readers as soon as the page is published. See [Using the Macro](using-the-macro.html) for the full configuration reference.

## Permissions

The app requires standard Confluence Cloud permissions. No additional Atlassian admin setup is needed beyond installing the app.

The **Admin Dashboard** is only accessible to Confluence site administrators. See [Admin Dashboard](admin-dashboard.html) for details.
